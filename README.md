# Welcome to FRW Community

**FRW - Free Resilient Web.**  
Open protocols. Open standards. Open access. No one owns it. No one controls it. Everyone can publish.

We're building a decentralized web that champions privacy, peer-to-peer tech, and digital rights. Powered by [FRW-Free-Web](https://github.com/frw-community/FRW-Free-Web), our flagship repo for distributed systems, Web3 tools, and anti-censorship innovation.
